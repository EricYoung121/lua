# add_input_string

用于在菜单中添加输入字符串的编辑框 上限 256 个字符

**代码示例：**
```lua
gui.add_input_string("添加一个字符串编辑框“，函数()

-这里填写执行的函数代码

结束)
```

组件相关函数

```lua
void set_text(std::string) -设置标题
 
std::string get_text() -取标题

std::string get_text(bool) -取编辑框内容
```
